Configuration

The Acme SDK reads configuration from a single file at the root of your project. Every key is optional and falls back to a sensible default.

Configuration is resolved once at startup and cached for the lifetime of the process.

Options

KeyTypeDefaultDescription
timeoutnumber8000Request timeout in milliseconds.
retriesnumber3How many times to retry a failed request.
baseUrlstringhttps://api.acme.devThe API origin to call.
cachebooleantrueWhether to cache idempotent responses.

After editing the file, restart the process for changes to take effect.
