# Memory distiller

Use this as a native memory rail.

## Distill

Separate:

- Stable identity and preferences.
- Project facts.
- Decisions made.
- Tasks completed.
- Open loops and blockers.
- Troubleshooting issue and solution pairs.

## Privacy

Do not save secrets, one-time codes, private credentials, or unnecessary personal data. Prefer short, durable facts over transcript summaries.

## Output

Return saved facts, skipped sensitive items, and open loops.
