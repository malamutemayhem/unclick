# Brand voice and guidelines

Use this when copy needs to sound like one brand, not many.

## Run

1. If guidelines exist, load them. If not, infer voice from the best existing examples.
2. Name the voice in a few traits and the tone shifts per context (sales, support, docs).
3. Keep a short do and do-not list: words to use, words to avoid, formatting rules.
4. Apply it to the draft; flag anything off-voice with a suggested rewrite.
5. Keep terminology and product names consistent throughout.

## Rule

Match the brand, not your own default style. When unsure, ask for one reference piece.

## Output

Return the on-brand copy plus a short note on the voice rules you applied.
