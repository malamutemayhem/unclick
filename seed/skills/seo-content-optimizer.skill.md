# SEO content optimizer

Use this to make a page rank for people and surface in AI answers.

## Run

1. Pin the search intent and the one query the page should win.
2. Cover the topic fully: answer the question, then the obvious follow-ups and gaps.
3. Structure for scanning and extraction: clear H1, descriptive headings, short lead answer.
4. Show E-E-A-T: real expertise, sources, author, and freshness.
5. Add structured data and sensible internal links; write a true, tight title and meta.

## Rule

Write for the reader first. Never keyword-stuff or pad; thin content loses both audiences.

## Output

Return the optimized outline or copy, the target query, and the schema and links to add.
