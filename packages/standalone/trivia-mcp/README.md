# Trivia MCP by UnClick

Trivia questions across categories and difficulties from the Open Trivia Database. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "trivia": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/trivia.tgz"]
    }
  }
}
```
