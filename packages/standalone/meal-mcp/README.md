# Recipes MCP by UnClick

Search recipes by name, ingredient, category, or area, with full instructions, from TheMealDB. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "meal": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/meal.tgz"]
    }
  }
}
```
