# Australia Post MCP by UnClick

Australia Post parcel tracking, postcode lookup, and delivery times.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "australiapost": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/australiapost.tgz"]
    }
  }
}
```

## Tools

- `auspost_track_parcel`
- `auspost_get_postcode`
- `auspost_delivery_times`
