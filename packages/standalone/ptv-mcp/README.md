# PTV MCP by UnClick

Melbourne / Victoria public transport timetables, departures, and disruptions (PTV).

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "ptv": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/ptv.tgz"]
    }
  }
}
```

## Tools

- `ptv_search`
- `ptv_departures`
- `ptv_disruptions`
- `ptv_stops_on_route`
- `ptv_route_directions`
