# MusicBrainz MCP by UnClick

Open music metadata: artists, releases, and recordings from MusicBrainz. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "musicbrainz": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/musicbrainz.tgz"]
    }
  }
}
```

## Tools

- `mb_search_artists`
- `mb_search_releases`
- `mb_search_recordings`
- `mb_get_artist`
- `mb_get_release`
