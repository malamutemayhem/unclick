## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "fpl": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/fpl.tgz"]
    }
  }
}
```
