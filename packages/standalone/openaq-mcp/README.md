# OpenAQ Air Quality MCP by UnClick

Global air quality measurements by location and country from OpenAQ. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "openaq": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/openaq.tgz"]
    }
  }
}
```
