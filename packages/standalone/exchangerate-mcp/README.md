# Exchange Rates MCP by UnClick

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "exchangerate": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/exchangerate.tgz"]
    }
  }
}
```
