# ESPN Sports MCP by UnClick

Live scores, news, and team info across NFL, NBA, MLB, NHL, and soccer from ESPN. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "espn": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/espn.tgz"]
    }
  }
}
```
