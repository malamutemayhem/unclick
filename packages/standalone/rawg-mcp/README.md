# RAWG Games MCP by UnClick

Search video games with detail, screenshots, genres, and upcoming releases from RAWG.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "rawg": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/rawg.tgz"]
    }
  }
}
```

## Tools
