# USGS Earthquakes MCP by UnClick

Recent earthquakes worldwide and by region, with detail, from USGS. No API key.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "usgs": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/usgs.tgz"]
    }
  }
}
```

## Tools

- `usgs_recent_earthquakes`
- `usgs_earthquake_detail`
- `usgs_earthquakes_by_region`
