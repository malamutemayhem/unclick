# Genius Lyrics MCP by UnClick

Search songs and artists and get song and artist detail from Genius.

> By UnClick. 180+ tools plus persistent agent memory in one install: https://unclick.world

## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "genius": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/genius.tgz"]
    }
  }
}
```

## Tools

- `genius_search`
- `genius_get_song`
- `genius_get_artist`
- `genius_artist_songs`
