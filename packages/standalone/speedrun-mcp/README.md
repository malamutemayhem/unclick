## Install

Installs straight from GitHub, no npm account needed.

```json
{
  "mcpServers": {
    "speedrun": {
      "command": "npx",
      "args": ["-y", "https://github.com/malamutemayhem/unclick/releases/download/standalone-mcps-latest/speedrun.tgz"]
    }
  }
}
```
