# AI agent harnesses and proof checks in UnClick

Last updated: 2026-06-09.

## Short answer

UnClick XPass is a family of AI agent harnesses. Each harness checks one kind of work and records readable evidence: PASS, FAIL, warning, or N/A with a reason.

Use this page for searches such as AI agent harnesses, AI QA harness, agent proof checks, SEOPass, GEOPass, TestPass, UIPass, UXPass, SecurityPass, and AI work receipts.

## Harnesses in the XPass family

- TestPass: checks whether work functions and has test/build proof.
- UIPass: checks whether a screen looks right.
- UXPass: checks whether a user can finish the journey.
- SecurityPass: checks common security risk signals.
- CopyPass: checks whether wording is clear.
- FidelityPass: checks exact-copy and source fidelity.
- LegalPass: checks risk language and legal-readiness signals.
- SlopPass: checks AI-code quality and sloppy implementation signals.
- CommonSensePass: checks whether the status, claim, or proof makes sense.
- SEOPass: checks whether search engines can read the public surface.
- GEOPass: checks whether AI answer engines can understand the public surface and evidence.
- FlowPass: checks whether the workflow can complete.
- RotatePass: checks credential and rotation handling signals.
- WakePass: checks whether stalled work has an owner or next action.
- CompliancePass: checks enterprise-readiness and compliance posture signals.

## What a harness proves

An XPass harness proves that a specific check was run and what evidence was observed. It does not prove that the work is perfect.

Good XPass output names:

- the target being checked
- which harness rows ran
- evidence links such as tests, PRs, screenshots, ledgers, or receipts
- rows marked PASS, FAIL, warning, or N/A
- honest reasons for skipped or not-applicable checks

## Why this matters for AI work

AI agents can produce confident status text without enough evidence. XPass turns that into a checklist and receipt workflow.

For public SEO and GEO work:

- SEOPass checks whether Google-style crawlers can read the page.
- GEOPass checks whether AI answer engines can understand the page, cite stable evidence, and avoid unsupported claims.
- Neither one guarantees rankings, AI Overview inclusion, or citations.
